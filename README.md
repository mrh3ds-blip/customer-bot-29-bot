# نیندژدژدژ

این ربات به صورت خودکار توسط ربات‌ساز ساخته شده است.

- نوع ربات: فیلم و عکس / آرشیو رسانه
- امکانات: آپلود فیلم و عکس، دسته‌بندی رسانه، پنل مدیریت رسانه، پنل مدیریت ساده، درگاه پرداخت آنلاین
- پشتیبانی: ننتدنیپپژپ

## پنل مدیریت داخل ربات
ادمین ربات با /start منوی مدیریت را می‌بیند و می‌تواند محصولات/خدمات، متن‌ها، سوال‌های فرم، کارت‌به‌کارت، لینک پرداخت و مرچنت زرین‌پال را تغییر دهد.

## نکته پرداخت آنلاین
مرچنت/API زرین‌پال داخل پنل ذخیره می‌شود و پرداخت آنلاین با callback خودکار فعال است. Callback: /payment/zarinpal/callback
